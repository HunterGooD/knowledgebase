Сущности

Ресурс базы знаний

Ресурс базы знаний - это любая уникально адресуемая часть информации содержащаяся в базе знаний. Ресурс может быть частью гипертекстового документа, электронным документом, изображением, мультимедийным файлом, коллекцией других ресурсов и пр.

Уникальный идентификатор ресурса

Символьная строка, позволяющая идентифицировать какой-либо ресурс базы знаний и его месторасположение. Состоит из пути и фрагмента разделенных символом #. Путь в свую очередь представляет собой иерархию местеположения ресурса которая указывается символами /.

Пример: products/loans/cashloans/bystrocredit#rates

Метаинформация

Метаинформация - информация о содержании, местоположении, способах и методах предоставления ресурса базы знаний. Описание ресурса производится путем указания: уникального адреса ресурса, имени ресурса, тегов отражающих его содержание, владельца ресурса, прав доступа, коллекции ссылок в которых участвует ресурс.

Метаинформация содержит сведения о источнике/ах, которые содержат информацию ресурсов базы знаний. Источники систематизируются по дате и времени создания источника. Наиболее рано созданные источник содержит первую версию информации ресурса. Последний из созданных источников содержит последнюю версию информации ресурса.

Метаинформация содержит сведения о подписках на события связанные с ресурсом. При наступлении события пользователи подписанные на них уведомляются системой.

Наименование ресурса

Наименование ресурса для восприятия пользователями базы знаний.

Тег

Ключевое слово (тег, метка) — идентификатор для категоризации, описания, поиска, содержащий слово или набор слов, представляющих содержание ресурса базы знаний.

Подписка

Информация о подписке пользователя/группы пользователей системы на событие связанное с ресурсом.

Ссылка

Ссылка - это идентифицированная и связь между двумя ресурсами. Первый из которых является ресурсом источником, а второй ресурсом назначением, соответственно ссылка содержит идентификаторы ресурса источника и ресурса назначения. Идентификация связи сопровождается указанием на тип этой связи.

Сведения об источнике

Сведения об источнике данных ресурса содержат информацию о самом источнике, дате и времени его создания, дате и времени начала действия.

Источник данных

URN: Source

Источник информации ресурса. Файл или веб сервис в котором содержится информация ресурса.

Веб сервис

Адрес в сети, предоставляющий информацию ресурса. Информация ресурса может находиться по адресам сервисов как внутри банка, так и за его пределами.

Файл

Файл — электронный носитель с зафиксированной на нём в любой форме информацией, предназначенный для передачи, использования и хранения. В файлах содержится информация ресурса/ресурсов.

Электронный документ

Электронный документ — файл с зафиксированной на нём в любой форме информацией в виде текста.

Гипертекстовый документ

Гипертекстовый документ - это текстовый документ содержащий XML разметку и гипертекстовые ссылки на ресурсы.

Гиперссылка

Гиперссылка - это ссылочный элемент гипертекстового документа предназначенный для использования пользователем базы знаний. Гиперссылка идентифицирует связь документа или его части с ресурсом базы знаний.

Коллекция ресурсов

Множество ресурсов объединенных по принципу схожести информации отраженной в их метаописании. Группировка коллекуий ресурсов осуществляется по данным фильтра.

Фильтр для поиска ресурсов

Фильтр для поиска ресурсов базируется на параметрах мета информации ресурсов и сведений о событиях связанных с этим ресурсами.

Каталог ресурсов

Иерархическое множество путей ресурсов. Служит для иерархической систематизации ресурсов базы знаний.

Избранное

Коллекция наиболее используемых пользователями базы знаний ресурсов.

Получается при использовании предустановленного фильтра для поиска коллекций ресурсов.

Действие пользователя

Информация о действиях пользователей в базе знаний. Служит для регистрации активности пользователей с последующей их обработкой и предоставлением информации о наиболее востребованных ресурсах, дате и времени их создания, актуализации и архивирования. Содержит информацию о пользователе, ресурсе , дате и времени совершения действия и его типе.

Коллекция действий пользователей

Коллекция записей действий пользователей отфильтрованная и отсортированная по пользователю и/или ресурсу и/или дате и времени действия и/или типу действия.

Администратор базы знаний

Пользователь базы знаний в обязанности которого входит обслуживание базы знаний, размещение, преобразование, метаописание и актуализацию ресурсов.

Пользователь базы знаний

Сотрудник банка использующий базу знаний для поиска и ознакомления с ее ресурсами.

Владелец ресурса

Владелец ресурса - сотрудник банка в обазанности которого входит создание ресурса, поддержание его в актуальном состоянии, передача указаний на его публикацию в базе знаний или архивирование.

Право доступа к ресурсу

Право на просмотр ресурса пользователем/группой пользователей базы знаний.
